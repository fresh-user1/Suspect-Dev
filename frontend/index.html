What Is This?

SUSPECTED is an experimental registry that attempts to map suspicious transaction patterns and developer wallet history in the Solana ecosystem. Not an investment tool, but rather a transparency infrastructure.

On-Chain Native

All data is sourced from the public Solana blockchain. No confidential data or insider information.

Crowdsourced

The community can submit transaction evidence (tx hash) which is then verified collectively, not centralized single authority.

Open Structure

Data structure is designed to be integrated by other wallets via API. Not a walled garden.

What's Different from Other Platforms?

Many scanners focus on token contracts (honeypot, mint authority). We focus on actor behavior — historical patterns of the same developer wallet recurring across different projects.

  • Cross-project tracking (same wallet across different tokens)
  • Focus on liquidity drain patterns, not just code audits
  • No "pay to remove" labels — absolute transparency
