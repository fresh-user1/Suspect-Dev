Apa Ini?

SUSPECTED adalah registry eksperimental yang mencoba memetakan pola transaksi mencurigakan dan histori wallet developer di ekosistem Solana. Bukan alat investasi, melainkan infrastruktur transparansi.

On-Chain Native

Semua data bersumber dari blockchain Solana yang publik. Tidak ada data rahasia atau insider information.

Crowdsourced

Komunitas dapat mengajukan bukti transaksi (tx hash) yang kemudian diverifikasi bersama, bukan sentralisasi otoritas tunggal.

Open Structure

Struktur data dirancang agar bisa di-integrasikan wallet lain via API. Bukan walled garden.

Bedanya dengan Platform Lain?

Banyak scanner yang fokus pada kontrak token (honeypot, mint authority). Kami fokus pada actorbehaviour — pola historis wallet developer yang sama terulang di berbagai project.

  • Tracking cross-project (wallet yang sama di token berbeda)
  • Fokus pada liquidity drain pattern, bukan sekadar code audit
  • Sistem reputasi validator terbuka (siapapun bisa jadi verifier)
  • Tidak ada "pay to remove" label — transparansi mutlak

Point & Incentive System

Mekanisme awal untuk membangun database: Kontributor yang memberikan bukti valid akan mendapatkan poin yang nantinya menjadi parameter alokasi reward.

FORMULA ALLOCATION (KONSEP AWAL)
(Total Poin Kontribusi × Weight) + (Avg Token Hold × Duration) = Allocation Score

*Detail matematis dan smart contract distribution masih dalam pengembangan. Tidak menjanjikan nilai moneter spesifik.

Cara Submit

1. Temukan tx hash penarikan likuiditas (liquidity removal)
2. Copy wallet address developer
3. Isi form dengan evidence link (Solscan)
4. Tunggu verifikasi 24-48 jam
5. Poin masuk otomatis setelah approved

Anti-Gaming

- Minimal holding token 7 hari untuk eligible airdrop
- Sybil resistance via on-chain activity check
- False report akan mengurangi reputasi (bukan poin)
- Multi-wallet detection otomatis

Development Roadmap

Proyek ini kompleks dan memerlukan fase pengembangan panjang. Tidak ada janji "moon" dalam 3 bulan.

Fase 01 — Q1 2025
MVP & Data Seeding In Progress
Landing page statis, form submission manual, kurasi data 50-100 kasus historis secara manual oleh tim inti.
  • Frontend HTML/CSS/JS (current)
  • Database MongoDB setup
  • Manual verification system
  • Community onboarding (Twitter/X)
Fase 02 — Q2 2025
Blockchair Integration Planned
Integrasi API Blockchair untuk otomatisasi tracing wallet dan deteksi pattern LP removal secara algoritmik.
  • Automated wallet clustering
  • Risk scoring algorithm v1
  • Browser extension beta (Phantom)
  • Point system smart contract (Solana)
Fase 03 — Q3 2025
Decentralized Verification Planned
Sistem staking token untuk menjadi validator. Mekanisme voting on-chain untuk status "Confirmed" vs "Disputed".
  • Validator staking mechanism
  • Dispute resolution court
  • Governance token launch (jika diperlukan)
  • API public untuk integrasi wallet
Fase 04 — Q4 2025
Cross-Chain & AI Analysis Research
Ekspansi ke jaringan EVM (Base, ETH) dan implementasi machine learning untuk deteksi pattern anomali otomatis.
  • Multi-chain support (Base, ETH)
  • ML-based anomaly detection
  • Mobile app (React Native)
  • Partnership dengan wallet besar
Fase 05 — 2026+
Autonomous Registry Vision
DAO sepenuhnya yang mengelola registry tanpa tim inti. Incentivized data correction dan self-sustaining economic model.
